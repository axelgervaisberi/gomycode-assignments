# DOM Project 2: Interactive Color Changer

This folder contains my solution for the Interactive Color Changer checkpoint.

### Structure
- **`index.html`**: Contains the semantic HTML structure, including the `.container`, the `#color-box`, and the `#change-color-btn`.
- **`style.css`**: Contains the styling to center the content horizontally, give the color box fixed dimensions and a default color, and make the button visually appealing with interactive hover/active effects.
- **`script.js`**: Contains the JavaScript logic. It waits for the `DOMContentLoaded` event, selects the elements using `getElementById()`, defines a `getRandomColor()` function that generates random Hex codes, and uses an event listener to apply the random color to the box when the button is clicked.

### How to Test
Simply double-click `index.html` to open it directly in your browser and click the "Change Color" button to watch the box change colors!

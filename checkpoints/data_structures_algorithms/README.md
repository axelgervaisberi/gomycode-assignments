# VisuAlgo Exploration

I spent some time on VisuAlgo testing out the interactive visualizations for various data structures and sorting algorithms. 

Here is a quick summary of my progress:

### What I tested and feel comfortable with:
I went through the visualizations for **Arrays**, **Queues**, and **Linked Lists**, as well as **Binary Search**. 

For sorting, I spent a lot of time on **Merge Sort**, which is definitely my favorite algorithm. I really like the "divide and conquer" technique it uses—breaking the array down into smaller halves makes a lot of logical sense to me.

### What I'm not familiar with yet:
There are two main topics I still haven't mastered:

1. **Hashmaps:** I haven't quite wrapped my head around these yet because the concept of hash functions and collision resolution (like chaining or probing) is a bit abstract. It feels more complicated than the simple pointers in linked lists or indexes in arrays, so I need to spend more time studying the math and logic behind it.

2. **Heap Sort:** I'm still struggling with this one because it requires a strong understanding of tree structures (specifically building and maintaining a heap). Since I haven't fully mastered binary trees and the "heapify" process, the actual sorting part of Heap Sort isn't clicking for me yet.

I'll be reviewing the documentation and practicing more with the visualizations for these two.

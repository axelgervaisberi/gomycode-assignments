# JavaScript Project 2: Functions Checkpoint

This folder contains my solutions for the JavaScript functions checkpoint. 

I've separated the solutions into three different files based on their category:
1. **`stringFunctions.js`**: Contains string manipulation functions (reverse, count characters, capitalize)
2. **`arrayFunctions.js`**: Contains array functions (find max/min, sum, filter)
3. **`mathFunctions.js`**: Contains mathematical functions (factorial, prime check, Fibonacci sequence)

I added some quick `console.log` tests at the bottom of each file so you can easily run them with Node.js and see the outputs to verify my problem-solving approach. I made sure to implement the logic clearly using fundamental programming constructs (like loops) rather than relying exclusively on built-in shorthand methods.

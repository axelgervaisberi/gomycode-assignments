# Insertion Sort

This is a JavaScript implementation of the insertion sort algorithm.

Insertion sort builds a sorted array one element at a time. It's similar to the way you might sort playing cards in your hands: you pick a card and place it in the correct spot among the cards you've already sorted.

The code uses 2 counters to work: `i` to keep track of the current element being sorted, and `j` to scan through the already sorted part of the array to find where the element belongs.
